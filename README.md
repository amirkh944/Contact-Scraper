# Contact Scraper - استخراج کننده اطلاعات تماس

اپلیکیشنی حرفه‌ای برای جمع‌آوری خودکار اطلاعات تماس (ایمیل و تلفن) از وبسایت‌های موجود در نتایج جستجوی گوگل.

## ویژگی‌های اصلی

✅ **جستجوی هوشمند**: استفاده از Google Custom Search Engine برای جستجوی دقیق
✅ **استخراج خودکار**: استخراج ایمیل و شماره تلفن از صفحات وب
✅ **رابط کاربری فارسی**: رابط‌ای زیبا و راستچین برای کاربران فارسی
✅ **صادرات چندگانه**: صادر کردن نتایج به فرمت‌های Excel، CSV و JSON
✅ **پشتیبانی ایرانی**: شناخت شماره‌های تلفن ایرانی
✅ **تنظیمات قابل سفارش**: مدیریت کامل تنظیمات از رابط کاربری

## نیازمندی‌ها

- Python 3.8+
- Windows 10 یا بالاتر

## نصب و راه���اندازی

### 1. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### 2. تنظیم Google Custom Search Engine

1. به [Google Cloud Console](https://console.cloud.google.com) مراجعه کنید
2. یک پروژه جدید ایجاد کنید
3. Google Custom Search API را فعال کنید
4. کلید API را ایجاد کنید
5. شناسه Custom Search Engine (CSE ID) خود را دریافت کنید

### 3. اجرای برنامه

```bash
python main.py
```

## استفاده

### مراحل بهکار

1. **وارد کردن تنظیمات**:
   - تب "⚙️ تنظیمات" را باز کنید
   - API Key و CSE ID را وارد کنید
   - بر روی "💾 ذخیره تنظیمات" کلیک کنید

2. **جستجو**:
   - تب "🔍 جستجو" را انتخاب کنید
   - کلمات کلیدی خود را وارد کنید
   - تعداد نتایج مورد نظر را مشخص کنید
   - بر روی "🚀 شروع جستجو" کلیک کنید

3. **مشاهده نتایج**:
   - تب "📊 نتایج" را باز کنید
   - نتایج یافت شده را مشاهده کنید

4. **صادر کردن**:
   - یکی از گزینه‌های صادرات را انتخاب کنید
   - نتایج در فرمت مورد نظر ذخیره می‌شوند

## فرمت‌های صادرات

### Excel
- فایل .xlsx با سبک‌های قابل دید
- ستون‌های: وبسایت، دامنه، لینک، ایمیل، تماس

### CSV
- فایل .csv قابل استفاده در تمام نرم‌افزار‌های جداول
- کدگذاری UTF-8

### JSON
- فایل .json برای پردازش برنامه‌ای
- ساختار کامل داده‌ها

## نکات مهم

⚠️ **محدودیت‌های API**:
- Google Custom Search API به صورت رایگان 100 جستجو در روز فراهم می‌کند
- برای جستجوهای بیشتر باید پلن پرداخت انتخاب کنید

⚠️ **احترام به robots.txt**:
- اپلیکیشن احترام به فایل robots.txt وبسایت‌ها نمی‌گذارد
- بهتر است از آن به‌صورت مسؤول‌انه استفاده کنید

⚠️ **داده‌های حساس**:
- از نتایج استخراج شده فقط برای مقاصد مشروع استفاده کنید

## حل مشکلات

### خطای "نتایج جستجو یافت نشد"
- API Key و CSE ID را بررسی کنید
- اتصال اینترنتی خود را کنترل کنید
- محدودیت API را بررسی کنید

### خطای "بارگذاری وبسایت"
- وبسایت شاید دسترسی را محدود کرده باشد
- حد زمانی را افزایش دهید
- VPN استفاده کنید

### نتایج خالی
- کلمات کلیدی بهتری انتخاب کنید
- تعداد نتایج را افزایش دهید
